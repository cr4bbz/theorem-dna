# Vision

Theorem DNA is a system for formal philosophy in which source material,
interpretation, formalization, proof checking, and derived corollaries are kept
in a single auditable knowledge ledger.

The system does not claim that philosophical prose can be translated without
human interpretation. Instead, it preserves the source text losslessly and records
every formalization decision as a versioned, reviewable object.

## Design rule

A corollary is accepted only when at least one configured proof assistant checks it.

## Epistemic rule

Validity is always relative to:

- source span
- interpretation
- logic profile
- context
- dependencies
- prover environment
